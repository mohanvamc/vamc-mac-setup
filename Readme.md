# Pre-req

Install Xcode

``` xcode-select --install ```

Install brew 

``` /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" ```


# Clone the Repo
 ```
 git clone https://github.com/mohanvamc/vamc-mac-setup.git

 ```

# Run the script
```
cd vamc-mac-setup
chmod +x setup-tools-mac.sh
./setup-tools-mac.sh

```
